# 현재 개발 상태

기준일: 2026-07-15

## 요약

- 핵심 사용자·관리자 기능은 파일럿 사용 가능한 수준으로 구현되었습니다.
- 관리자 웹과 API는 운영 배포되어 있습니다.
- Android APK는 빌드되지만 현재 release가 debug 키로 서명되므로 스토어 출시용은 아닙니다.
- 상용 출시 전에는 DB 마이그레이션 대조, 결제·환불·정산 E2E, 운영 서명과 CI가 필요합니다.

## 현재 작업 트리 주의

- `apps/customer/ios/**` 여러 파일이 줄바꿈 변환으로 보이는 수정 상태입니다. 검토 전 일괄 커밋하지 않습니다.
- 결제 스키마는 새 개발 DB에 `0001~0030` 전체 적용됐으며 이전 PG 호환 스키마를 승계하지 않습니다.
