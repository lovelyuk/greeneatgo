# 그린잇(GreenIt) 그린 리브랜딩 디자인

- 작성일: 2026-07-08
- 상태: 승인됨 (구현 진행)

## 배경

세 개의 UI 표면(관리자 웹, 고객 앱, 이메일 인증 페이지)이 모두 브랜드명은 이미
"그린잇"이지만, 색감과 카피는 따뜻한 주황색 "출출박스" 스타일로 되어 있다.
브랜드 정체성("그린잇")에 맞게 **신선한 새싹 그린 + 건강한 먹거리 톤**으로
리브랜딩한다.

## 컨셉

**"그린하게 먹다, 건강하게 잇하다"**

구조·레이아웃·기능·로직은 그대로 두고 **색 토큰과 핵심 카피만** 교체하는
순수 시각/문구 작업이다.

## 색 토큰 매핑 (세 곳 공통)

세 파일이 동일한 웜 팔레트를 공유하므로 아래처럼 1:1 치환한다.

| 역할 | 현재 (주황) | 변경 (그린) |
|---|---|---|
| 메인 / Primary | `#FF7A1A` | `#2FB865` |
| 보조 / 그라데이션 끝 | `#FFA629` | `#7BD88F` |
| 라임 포인트 | (없음) | `#A7E86B` |
| 배경 | `#FFF7E8` 크림 | `#F3FBF4` 민트크림 |
| 카드 | `#FFFCF4` | `#FCFEFC` |
| 본문 진한 글자 (ink) | `#23190F` | `#14351F` |
| 제목 / 로고 (cocoa) | `#4A2A14` 갈색 | `#1E5631` 숲그린 |
| 보조 글자 (갈색들) | `#805A39` · `#8B6544` · `#7A5637` · `#9A7656` · `#B39678` | `#5C7A66` 그린그레이 (톤에 맞춰 근사치) |
| 테두리 / 라인 | `#FFDEAA` / `#FFDFA8` / `#FFE6BB` / `#FFECD0` | `#CDEBD5` 계열 |
| 배경 그라데이션 · 블롭 | 주황 계열 | 그린 계열 |

성공/에러 alert(초록/빨강)와 승인/거절 버튼 색은 이미 적절하므로 유지한다.

## 핵심 카피 교체 (스낵 → 건강 먹거리)

| 위치 | 현재 | 변경 |
|---|---|---|
| 고객앱 로고 부제 | 회사 식대가 맛있어지는 시간 | **그린하게 먹다, 건강하게 잇하다** |
| 고객앱 로그인 버튼 | 따뜻한 한 끼 시작하기 | 그린한 한 끼 시작하기 |
| 고객앱 히어로 | 출출할 땐 바로 결제 | 건강하게, 바로 결제 |
| 고객앱 홈 부제 | ...출출할 땐 회사 식대로 간편하게 해결해요 | 그린하게 챙기는 오늘 한 끼 |
| 관리자 히어로 h1 | 회사 식대를 간식박스처럼 가볍게 운영해요 | 그린하게 먹고 건강하게 잇는 회사 식대 |
| 관리자 부제 | ...출출박스 스타일의 카드 대시보드... | ...그린잇 스타일의 카드 대시보드... |
| 카테고리 이모지 | 🥪 간식 · 🍊 | 🥗 샐러드 · 🍃 로 일부 교체 |

나머지 화면 문구(입력 안내, 상태 메시지, 영어 pill 라벨 등)는 유지한다.
