## 공통 로그인 안내

- 관리자 웹 주소: `https://greeneatgo.vercel.app/`
- 관리자 웹은 로그인한 계정의 역할에 따라 업체관리자 또는 식당관리자 화면이 자동으로 열립니다.
- 사용자 앱은 이메일과 비밀번호로 로그인합니다.
- 관리자·사용자 모두 비밀번호는 6자 이상이어야 합니다.
- 공용 PC나 다른 사람의 휴대폰에서는 사용 후 반드시 로그아웃하세요.

## 문서 표기

- **굵은 글씨**: 화면에서 누르는 버튼이나 메뉴
- `코드 형태`: 입력 예시 또는 주소
- ‘일반 사용자’: 개인이 식권 패키지를 구매해 사용하는 고객
- ‘회사 직원’: 소속 회사의 장부·지원 정책으로 결제하는 직원
- ‘업체관리자’: 회사의 직원과 식대 정책을 관리하는 담당자
- ‘식당관리자’: 식당, 상품, 업체 거래와 정산을 관리하는 담당자

## 운영 전 확인이 필요한 항목

일부 기능은 서버 환경변수, 데이터베이스 마이그레이션 또는 외부 서비스 설정에 따라 활성화됩니다.

- 이메일 초대 발송: 이메일 발송 서비스 설정 필요
- 앱 푸시 알림: Firebase 설정 및 사용자의 알림 권한 필요
- 일반/지원 식권 카드결제: 토스페이먼츠 설정 필요
- 상품·메뉴 이미지: 이미지 저장소 설정 필요
- 상품 이벤트, 알림, 일괄등록, 기간 정산: 관련 DB 마이그레이션 필요

문서와 실제 화면이 다르면 브라우저 또는 앱을 최신 버전으로 갱신한 뒤 다시 확인하세요.
