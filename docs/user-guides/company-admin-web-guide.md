# 업체관리자 웹 사용설명서

## 1. 업체관리자 웹이란?

업체관리자 웹은 회사의 식대 운영 담당자가 다음 업무를 처리하는 화면입니다.

- 직원 가입 요청 승인·거절
- 회사 초대코드 전달
- 직원 일괄등록
- 직원 정보와 포인트 관리
- 직원별 식대·포인트 이용내역 확인
- 중식·석식 사용시간 설정

관리자 웹 주소: `https://greeneatgo.vercel.app/`

---

## 2. 로그인 및 화면 구성

### 로그인

1. 관리자 웹에 접속합니다.
2. 이메일과 비밀번호를 입력합니다.
3. **운영 시작하기**를 누릅니다.

로그인 후 업체관리자 권한이 확인되면 업체관리자 대시보드가 열립니다.

### 초대링크로 처음 가입하기

1. 식당관리자에게 받은 초대링크를 엽니다.
2. 권한, 초대 상태, 만료일을 확인합니다.
3. 이메일, 이름, 6자 이상의 비밀번호를 입력합니다.
4. **새 계정으로 가입하고 초대 수락**을 누릅니다.

초대에 이메일이 지정돼 있으면 다른 이메일로 변경할 수 없습니다.

### 상단 버튼

- **새로고침**: 직원, 가입 요청, 정책 등 최신 정보를 다시 불러옵니다.
- **로그아웃**: 현재 관리자 계정에서 나갑니다.

---

## 3. 대시보드 요약

상단 카드에서 다음 수치를 확인할 수 있습니다.

- **가입 요청**: 아직 승인하지 않은 직원 수
- **직원**: 등록된 직원과 가입 대기 직원 수

가입 요청 수가 있으면 화면 아래의 **가입 요청 승인** 영역을 확인하세요.

---

## 4. 관리자 정보 변경

**로그인 정보** 패널에서 이메일과 관리자 이름을 확인합니다.

1. 관리자 이름 옆 톱니바퀴를 누릅니다.
2. 필요한 항목을 변경합니다.
   - 관리자 이름
   - 새 비밀번호
   - 새 비밀번호 확인
3. **변경사항 저장**을 누릅니다.

로그인 이메일은 이 화면에서 변경할 수 없습니다. 비밀번호를 바꾸지 않을 때는 새 비밀번호 칸을 비워두세요.

---

## 5. 회사 초대코드 사용

직원이 회사에 가입하려면 회사 초대코드가 필요합니다.

1. 초대코드 패널에서 **복사**를 누릅니다.
2. 복사한 코드를 직원에게 전달합니다.
3. 직원은 앱에서 **회사 장부 가입 요청**을 선택하고 코드를 입력합니다.
4. 직원의 요청이 업체관리자 화면의 **가입 요청 승인** 영역에 표시됩니다.

초대코드는 회사 내부 직원에게만 전달하세요.

---

## 6. 직원 가입 요청 승인·거절

화면 아래 **가입 요청 승인** 영역에서 대기 중인 직원을 확인할 수 있습니다.

### 승인

1. 요청자의 이름과 요청일을 확인합니다.
2. **승인**을 누릅니다.
3. 승인된 직원은 등록 직원목록에 활성 직원으로 표시됩니다.

### 거절

1. **거절**을 누릅니다.
2. 거절 사유를 입력합니다.
3. 확인하면 요청이 거절됩니다.

잘못된 회사로 신청한 직원이나 확인되지 않은 직원은 승인하지 마세요.

---

## 7. 식대 사용시간 설정

직원 결제를 중식·석식 시간으로 제한할 수 있습니다.

### 시간 제한 사용

1. **사용시간 제한 켜기**를 선택합니다.
2. 중식 시작·종료 시간을 입력합니다.
3. 석식 시작·종료 시간을 입력합니다.
4. **식대 사용시간 저장**을 누릅니다.

### 시간 제한 해제

1. **사용시간 제한 켜기** 선택을 해제합니다.
2. **식대 사용시간 저장**을 누릅니다.

해제하면 직원은 시간 제한 없이 사용할 수 있습니다. 시작 시간은 종료 시간보다 이르게 입력하세요.

---

## 8. 등록 직원목록

직원목록에서 다음 정보를 확인할 수 있습니다.

| 항목 | 설명 |
|---|---|
| 상태 | 초대대기, 사용중 등 현재 상태 |
| 부서 | 직원 소속 부서 |
| 이름 | 직원 이름 |
| 사번 | 회사 사번 |
| 전화번호 | 등록 연락처 |
| 포인트 잔액 | 현재 사용 가능한 포인트 |
| 이번 달 이용액 | 이번 달 식대 이용 금액 |
| 최근 이용일 | 가장 최근 사용 일시 |

`초대대기` 직원은 아직 앱 가입이 끝나지 않은 상태이므로 이용내역과 포인트 관리 기능을 사용할 수 없습니다.

---

## 9. 직원 일괄등록

여러 직원을 엑셀 또는 CSV로 한 번에 등록할 수 있습니다.

1. 직원목록에서 **+ 직원 일괄등록**을 누릅니다.
2. **엑셀 양식 다운로드**를 누릅니다.
3. 양식의 헤더 순서를 바꾸지 않고 직원 정보를 입력합니다.
4. 작성한 `.xlsx` 또는 `.csv` 파일을 업로드합니다.
5. 정상 행과 오류 행을 확인합니다.
6. 오류가 있으면 **에러 목록 다운로드**로 내용을 확인합니다.
7. **정상 N건만 등록 확정**을 누릅니다.

### 작성 시 주의

- 한 파일에 최대 500명까지 작성하세요.
- 파일은 최대 10MB입니다.
- 사번과 전화번호가 기존 직원과 중복되지 않도록 확인하세요.
- 등록된 직원은 먼저 `초대대기` 상태가 되며, 직원이 앱 가입을 마쳐야 활성화됩니다.

---

## 10. 직원 정보 및 포인트 관리

활성 직원 행의 톱니바퀴를 눌러 **직원 관리** 창을 엽니다.

### 변경 가능한 정보

- 부서
- 이름
- 사번
- 전화번호

### 포인트 충전

1. **즉시 포인트 충전**에 추가할 금액을 입력합니다.
2. **저장**을 누릅니다.

### 목표 잔액으로 조정

1. **조정 후 목표 잔액**에 최종 잔액을 입력합니다.
2. **저장**을 누릅니다.

충전액과 목표 잔액을 동시에 입력하면 충전 후 목표 잔액으로 다시 조정되므로, 일반적으로 둘 중 하나만 사용하세요.

---

## 11. 직원 이용내역 확인

활성 직원 행에서 **이용내역**을 누릅니다.

### 식대 이용내역

- 일시
- 식당
- 상품 또는 거래 내용
- 거래 구분
- 금액

### 포인트 내역

- 일시
- 충전·사용·조정 구분
- 포인트 증감액
- 변경 후 잔액

목록에는 최근 내역이 표시됩니다. 직원이 자신의 앱에서 보는 최근 이용과 관리자 화면의 전체 관리 내역은 표시 범위가 다를 수 있습니다.

---

## 12. 권장 운영 순서

1. 업체관리자 계정으로 로그인
2. 관리자 이름과 회사 초대코드 확인
3. 필요하면 직원 일괄등록
4. 직원들에게 앱 설치와 초대코드 전달
5. 들어온 가입 요청 승인
6. 식대 사용시간 설정
7. 직원별 이용액과 최근 이용일 확인
8. 필요할 때 직원 정보·포인트 수정
9. 이용내역 확인
10. 업무 완료 후 로그아웃

---

## 13. 자주 묻는 질문

### 직원이 목록에 보이지 않아요

- 직원이 올바른 회사 초대코드로 요청했는지 확인하세요.
- **새로고침**을 누르세요.
- 일괄등록 직원이라면 `초대대기` 상태인지 확인하세요.

### 직원이 결제할 수 없어요

- 직원 상태가 `사용중`인지 확인하세요.
- 식대 사용시간 제한에 걸리지 않았는지 확인하세요.
- 월 한도, 계약 상태 및 식당 QR이 올바른지 확인하세요.

### 포인트를 잘못 충전했어요

직원 관리에서 **조정 후 목표 잔액**을 사용해 원하는 최종 잔액으로 맞추세요. 포인트 내역에는 조정 기록이 남습니다.

### 일괄등록 버튼이 비활성화돼요

관련 데이터베이스 기능이 아직 운영 환경에 적용되지 않은 상태일 수 있습니다. 시스템 관리자에게 문의하세요.
